# i3 でプロジェクタを使いたい

## 作戦1. xfce に寄生する (chromebook環境)


## 作戦2. xrandrを使う

* [i3 User’s Guide](https://i3wm.org/docs/userguide.html#multi_monitor)
* [xrandr - ArcWiki](https://wiki.archlinux.jp/index.php/Xrandr)

```
$ xrandr 
```
で、ディスプレイ名を得る。その後

```
$ xrandr --output VGA1 --auto --left-of LVDS1
```

で設定すればいい。
